Install ACEPEAK on Your Phone
Make ACEPEAK a real app on your home screen. Works offline. Takes 2 minutes.
Step 1 — Host the file (free, permanent URL)
1
Go to github.com — create a free account.
2
Create a new repository named acepeak. Set to Public.
3
Upload your acepeak.html file renamed to index.html.
4
Settings → Pages → Deploy from branch → main → / root → Save.
5
Your permanent URL: https://[username].github.io/acepeak
Step 2 — Add to Home Screen (iPhone — must use Safari)
1
Open your URL in Safari.
2
Tap the Share button → "Add to Home Screen" → Add.
Step 2 — Add to Home Screen (Android — Chrome)
1
Open URL in Chrome → three-dot menu → "Add to Home screen".
⚠ Export a backup first before moving to a hosted URL. Restore it after opening the new URL.
